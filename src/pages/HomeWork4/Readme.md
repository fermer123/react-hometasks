# Задание.

## Требуется реализовать хук useRef, используя уже изученные хуки (useState, useEffect, useMemo, useCallback, useReducer).

## Инструкция.

### Чтобы проверить работоспособность хука, можно поместить на страницу кнопку, зацепиться за неё вашим хуком, и сделать так, чтобы при клике она меняла свой цвет, и при этом компонент, внутри которого эта кнопка находится, не перерендеривался.

## Примечания:

1. Чем компактнее написан код - тем лучше.
